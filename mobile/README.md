# Reverb256 Portfolio Mobile App

A React Native mobile companion for the Reverb256 portfolio with consciousness-driven features and cross-platform capabilities.

## Features

- **Portfolio Companion**: Complete mobile version of the Reverb256 portfolio website
- **Project Showcase**: Interactive display of all portfolio projects with live updates
- **Consciousness Integration**: AI-driven insights and project status monitoring
- **Cross-Platform Support**: iOS and Android with consistent user experience
- **Offline Capabilities**: Cached content for browsing without internet connection
- **Real-time Updates**: Live project status and AI system monitoring
- **Performance Optimization**: Optimized for mobile performance and battery life

## Architecture

### Mobile Integration
- **Portfolio Synchronization**: Real-time sync with web portfolio
- **Mobile-First Design**: Touch-optimized interface for mobile browsing
- **Offline Storage**: AsyncStorage for cached content and preferences
- **Push Notifications**: AI system alerts and project updates

### Screens
- **HomeScreen**: Portfolio overview and featured projects
- **ProjectsScreen**: Interactive project gallery with details
- **AboutScreen**: Developer information and philosophy
- **ContactScreen**: Mobile-optimized contact options
- **SettingsScreen**: App preferences and AI integration settings

## Getting Started

### Prerequisites
- Node.js 16+
- React Native CLI
- Android Studio or Xcode
- Access to the main portfolio API endpoints

### Installation

```bash
cd mobile
npm install
```

### Running the App

#### Android
```bash
npm run android
```

#### iOS
```bash
npm run ios
```

### Development
```bash
npm start
```
